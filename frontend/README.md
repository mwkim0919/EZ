## Installation

```sh
yarn
```

## Development

```sh
yarn start
```

```sh
# If you're developing a generic / reusable component like Button, use react-styleguidist. Refer to src/components/Button for an example.
yarn styleguide
```

## Testing

```sh
# Run all tests just once
yarn run test

# If you want to watch tests, run
yarn run test:watch
```

## Deployment

```sh
yarn run build
```
